Support Vector Machine

Support Vector Machine

介绍

  • 有监督学习模型
  • 通常用来进行模式识别、分类以及回归分析
  • 超平面分类器
  • 针对线性可分情况进行分析，对于线性不可分的情况，通过使用非线性映射算法将低维输入空间线性不可分的样本转化为高维特征空间使其线性可分，从而使得高维特征空间采用线性算法对样本的非线性特征进行线性分析成为可能。
  • 它基于结构风险最小化理论之上在特征空间中构建最优超平面，使得学习器得到全局最优化，并且在整个样本空间的期望以某个概率满足一定上界
